Click - Move points
CTRL + click - Create new point
CTRL + Shift + click - Create new point in a new curve
DELETE - Delete last point
C - De Casteljau skeleton visualization
S - Split curve on current t value
M - Merge selected curve with the closest curve to it, make them into one curve
K - Make selected curve differentiable with the closest curve to it, keep them as two different curves
F - Make selected curve continuous with the closest curve to it, keep them as two different curves
N - Go to next curve
CTRL + Z - Undo
CTRL + Y - Redo
