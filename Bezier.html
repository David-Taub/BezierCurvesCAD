Click - Move points
CTRL + click - Create new point
CTRL + Shift + click - Create new point in a new curve
DELETE - Delete last point of selected curve
CTRL + DELETE - Delete selected curve
C - De Casteljau skeleton visualization
S - Split curve on current t value
M - Connect selected curve with the closest curve to it, 0 derivative continuous
K - Connect selected curve with the closest curve to it, 1 derivative continuous
O - Connect selected curve with the closest curve to it, 2 derivative continuous
N - Go to next curve
CTRL + Z - Undo
CTRL + Y - Redo
